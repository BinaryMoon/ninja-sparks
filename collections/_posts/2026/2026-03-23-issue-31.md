## Slow Roads

[Slow Roads][slowroads-link] is an endless driving simulator. You drive along a road that is procedurally generated, with no goal beyond the joy of driving.

{% include embed-browsershots.html url="https://slowroads.io/" %}

It's a relaxing experience with a nice soundtrack and a zen like aura.

You can change the vehicle (car, bus, weird bike thing), the time of day, weather, environment, and what the roads are like (straight, windy, normal).

{% include embed-image.html image="/images/updates/2026/slowroads-1.png" alt="Slow Roads Gameplay at night" %}

Having made some smaller procedural things in the past, I can really appreciate how much effort it must have been to make this and to make something that is both relaxing to play, and visually interesting. Plus it does appear to be genuinely endless. At one stage I tried driving off road and it just kept on going, across a lake and through forests, so even away from the main path there is still a lot to see.

{% include embed-image.html image="/images/updates/2026/slowroads-2.png" alt="Driving a bus in Slow Roads" %}

Besides the ground and some fences and walls there's minimal collision in the game so you can happily drive through trees and lakes and just enjoy the vibes.

There's a Steam version of the game in progress which appears to have more varied environments, and vehicles, but the browser version is free and has a lot of content already, so it's worth checking out.

{% include embed-image.html image="/images/updates/2026/slowroads-3.png" alt="Slow Roads in a lush forest" %}

**[Try Slow Roads][slowroads-link]**

## Sparked Discoveries

Here’s a handful of links I found intriguing recently:

### [Web 1.0 Hosting](https://web1.0hosting.net/)

{% include embed-browsershots.html url="https://web1.0hosting.net/" %}

Web 1.0 hosting is a web hosting service that allows you to create and host a simple website using HTML, CSS, and JavaScript. The design of the site is very retro, reminding me of the early days of the web (the late 90s). It's a fun, cheap (often free) way to get a simple website up and running. It's also privacy focused with no ads, cookies or data collection, which is refreshing in the modern web landscape. There is a paid option that gives you more space, $1 for 1GB of storage, but the free option is very generous and perfect for a simple personal site or project. I like that they have WebChat, a forum, and an IRC channel giving you that old-school web community feel.

### [Hermit Hire](https://hermithire.com/)

{% include embed-browsershots.html url="https://hermithire.com/" %}

A website that connects landowners with hermits who are looking for a place to live. I think it's a parody site, but at the same time you can both apply to be a hermit, or to hire a hermit. Either way it's a bit of fun.

### [Tiny Snake](https://patorjk.com/games/subpixel-snake/)

{% include embed-browsershots.html url="https://patorjk.com/games/subpixel-snake/" %}

This is amazing, a version of th classic game snake where the snake is made up of subpixels on your screen. For those who don't know each pixel on your screen is made up of three subpixels (red, green and blue) that combine to create the full colour. In this game, the snake is made up of these subpixels, so it can be very very small. The game is tiny, but fortunately there's an option to make it bigger if you want so that you can actually see what's going on. It's a clever idea but absolutely pointless, but that's kind of why I like it. The developer also has a video explaining [how it works](https://www.youtube.com/watch?v=iDwganLjpW0).


### [Dialed](https://dialed.gg/)

{% include embed-browsershots.html url="https://dialed.gg/" %}

This is a simple game to see how well you can remember colours. You are shown a colour for 5 seconds and then you have to try to recreate it using a colour picker. It's a simple concept but it's quite fun and addictive. I did a lot better than I expected given I'm colour blind. I guess it doesn't matter what colour I see as long as I can reproduce it. I found it quite easy to remember the basic colour but getting the brightness right was a lot harder.
