## Drawing Timer

I said at the start of the year that I want to get better at art and I've found a new way to do it. Practicing drawing through timed excercises is a common technique, and I linked to some resources for that. However all of those resources are themed around figure drawing, and I want to practice specific types of art.

{% include embed-image.html image="/images/updates/2026/drawing-timer-settings.png" caption="Drawing Timer Settings" %}

Then last week I found [Drawing Timer](https://drawingtimer.com/), which is a free web app that lets you set up your own drawing exercises using Pinterest boards. You can set a board name, that you have compiled, and then the drawing times, and it will automatically switch to the next one when the time is up. It's a really simple idea, but it's perfect for what I want to do.

{% include embed-image.html image="/images/updates/2026/drawing-timer-practice.png" caption="Drawing Timer in action" %}

I'm now considering making a simple version of this that works on local images so I can use it for my own practice based on the library of images I've collected over the years.

**[Try Drawing Timer](https://drawingtimer.com/)**

## PicoCAD 2 Feature Trailer

{% include embed-youtube.html video="tr-mW9gcito" youtube_thumbnail="https://i.ytimg.com/vi/tr-mW9gcito/0.jpg" %}

I've mentioned [PicoCAD]({% link _library/app/2025-02-10-picocad.md %}) before and now, we have the trailer for version 2.

The new version includes a host of improvements. Where the old app was made with Pico-8 the new version is actually made with Love2D which is a more powerful system allowing more flexibility and better performance. New features include a built in texture editor, a new animation system, and object based scenes allowing for more complex models. It also has a new palette system allowing us to use different colours to the limited Pico-8 palette.

[Watch the video](https://www.youtube.com/watch?v=tr-mW9gcito)

## Sparked Discoveries

Here’s a handful of links I found intriguing recently:

### [DOOM on an Oscilloscope](https://front-end.social/@html5test/115967227563353825)

{% include embed-browsershots.html url="https://front-end.social/@html5test/115967227563353825" %}

Devs love to put DOOM on anything with a screen, but this is a new one. Niels Leenheer has managed to get [Doom running on an oscilloscope](https://front-end.social/@html5test/115967227563353825
), which is a device used to display electrical signals. It's a really impressive feat of engineering and programming, and it's amazing to see what people can do with old technology. This is a video and write up on Mastodon, so if you're into programming it's worth checking out to get some tips on how he did it. If you're not into programming, it's still worth checking out just to see the end result.

<!-- INCLUDE SHARE -->

### [Brickset Lego Catalogues](https://brickset.com/library/catalogues)

{% include embed-image.html image="/images/updates/2026/brickset.png" %}

I've used Brickset for years but for some reason I never noticed that they have a whole section of scanned Lego catalogues. It's really interesting to see how the sets have evolved over the years, and it's a great resource for anyone interested in Lego history. Plus I even recognised some of the catalogues from my childhood.

### [The Scale of Life](https://www.thescaleoflife.com/)

{% include embed-browsershots.html url="https://www.thescaleoflife.com/" %}

It's a simple website but there's soemthing about this that really puts thing into perspective. The website looks at the time you arrive at the site, and then shows you everything that has happened since then. In the time I've typed this sentence for example there have beeb 158 births. The first page is interesting but make sure to browse the different categories because there's all sorts of data there to compare yourself with. It's a nice reminder that we are all part of a much bigger picture, and that our time on this planet is just a small blip in the grand scheme of things.
